# iot-home-automation

I used Following Technology :
1) LAMP
2) Python CGI 

Hardware :
1) Raspberry Pi 3 MODEL B - 1
2) Bread Board - 1
3) Relay Switch - 1
4) Ultrasonic Sensor - 1
5) Jumper wire M2F - 10
6) Resister 330Ω and 470Ω
